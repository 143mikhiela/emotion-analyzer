# Emotion Analyzer (NLP + Cognitive Science)

## Overview
This project analyzes emotional tone in text using Natural Language Processing (NLP) and maps it to psychologically meaningful categories.

## Features
- Sentiment analysis using TextBlob
- Custom emotion classification (stress, growth, etc.)
- Data visualization with matplotlib

## Motivation
Basic sentiment analysis often misses emotions like anxiety or feeling “stuck.”  
This project improves accuracy by combining sentiment scores with cognitive interpretation.

## Tools Used
- Python
- TextBlob
- pandas
- matplotlib
